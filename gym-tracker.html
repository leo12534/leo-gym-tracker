How to deploy

1. Go to script.google.com → New project
2. Paste the GymTracker_AppsScript.gs code
3. Click Deploy → New deployment
4. Type: Web app
5. Execute as: Me
6. Who has access: Anyone
7. Click Deploy → copy the URL → paste above
